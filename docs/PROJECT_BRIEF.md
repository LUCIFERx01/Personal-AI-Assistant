# Project Brief: Alpha Personal AI Assistant

## Problem
Users need a practical assistant for daily planning, explanations, and lightweight coaching without complex setup.

## Solution
Alpha provides chat, image explanation, memory, tasks, and check-in coaching in a local-first FastAPI + web app stack.

## Business value
- Increases daily task clarity and follow-through
- Supports explainability and coaching use cases
- Local-first approach reduces data exposure risk

## Technical highlights
- FastAPI backend with structured endpoints
- Multimodal AI flows (text + image)
- SQLite persistence for memory/tasks/check-ins
- Security-first local runtime constraints
